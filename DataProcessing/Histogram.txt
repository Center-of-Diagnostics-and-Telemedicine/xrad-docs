Гистограммы
(проект)

Гистограмма показывает количества элементов контейнера, значения которых лежат
в определенных диапазонах.

Гистограмма есть дискретная функция, значение i-го элемента которой равно
количеству элементов исходного контейнера, значения которых лежат в пределах
i-го диапазона.

Предлагается использовать следующие системы координат гистограмм:
- (double) origin, (double) step, (size_t) size:
	- origin задает координаты центра первого интервала гистограммы,
	- step задает шаг (он же размер) интервалов,
	- size задает количество элементов гистограммы;
- (range1_F64) roi, (size_t) size:
	- roi задает границы диапазона гистограммы,
	- size задает количество элементов гистограммы.

Переход между системами координат происходит по следующим формулам:

roi.p1() = origin - 0.5 * step
roi.p2() = origin + (size - 0.5) * step
